Contexto

Detalhes sobre o espaço de problema, justificativas e os objetivos do projeto.

Problema
Nesse momento você deve apresentar o problema que a sua aplicação deve resolver. No entanto, não é a hora de comentar sobre a aplicação. Descreva também o contexto em que essa aplicação será usada, se houver: empresa, tecnologias, etc. Novamente, descreva apenas o que de fato existir, pois ainda não é a hora de apresentar requisitos detalhados ou projetos.

O problema que a aplicação enfrentará é a vulnerabilidade dos idosos aos golpes na internet devido à falta de familiaridade com o mundo digital. Muitos idosos não estão familiarizados com as nuances da segurança online, o que os torna alvos fáceis para que golpistas ou pessoas maliciosas se aproveitem dessa falta de conhecimento. Além disso, a falta de segurança dos idosos na internet representa um desafio adicional, já que muitos deles podem precipitadamente compartilhar informações pessoais sensíveis ou serem vítimas de phishing, fraudes financeiras e outros tipos de crimes cibernéticos.

Objetivos
Aqui você deve descrever os objetivos do trabalho indicando que o objetivo geral é desenvolver um software para solucionar o problema apresentado acima. Apresente também alguns (pelo menos 2) objetivos específicos dependendo de onde você vai querer concentrar a sua prática investigativa, ou como você vai aprofundar no seu trabalho.

Nosso objetivo é criar um site destinado a idosos, onde eles possam encontrar informações de seu interesse e aprender a se proteger contra golpes online. O site terá seções explicativas sobre temas como Educação Digital, Verificação de Links e Emails, Filtros de Conteúdo, Compartilhamento Seguro de Informações, além de outras funcionalidades.

Justificativa
Descreva a importância ou a motivação para trabalhar com esta aplicação que você escolheu. Indique as razões pelas quais você escolheu seus objetivos específicos ou as razões para aprofundar em certos aspectos do software. O grupo de trabalho pode fazer uso de questionários, entrevistas e dados estatísticos, que podem ser apresentados, com o objetivo de esclarecer detalhes do problema que será abordado pelo grupo.

Identificamos que por conta de uma falta de educação digital, de forma ingênua, muitos idosos acabam se tornando vítimas de ataques digitais de golpistas, muitas vezes acarretando em prejuízos financeiros e morais.
A SOFIA nasceu para suprir esse problema da falta de informação sobre segurança digital, trazendo uma plataforma intuitiva, de forma á informar e evitar que mais pessoas se tornem vítimas desses ataques mais comuns.

Como posso acessar a aplicação?
Você pode acessar a aplicação através do link fornecido pelo administrador do sistema. Certifique-se de ter as credenciais de login corretas.
Quais são os requisitos de sistema para usar a aplicação?
A aplicação pode ser usada em qualquer navegador moderno como Chrome, Firefox, ou Edge. Recomendamos uma conexão de internet estável para um desempenho ótimo.
Como são feitas as atualizações no site SOFIA?
As atualizações do site SOFIA são gerenciadas pela equipe de desenvolvimento. Os usuários não precisam realizar nenhuma ação para atualizar o site.
O site SOFIA pode ser integrado com outras ferramentas ou plataformas?
Atualmente, o site SOFIA não oferece integrações com outras ferramentas ou plataformas, pois é uma plataforma independente focada em fornecer informações e recursos diretamente aos usuários.
Quem desenvolveu o site SOFIA?
O site SOFIA foi desenvolvido por uma equipe dedicada de profissionais que utilizam tecnologias como Bootstrap e JSON Server para criar uma experiência de usuário eficiente e segura.
